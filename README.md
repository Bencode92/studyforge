## Fonctionnalites

- **Import PDF/Texte** → IA verifie et structure
- **Enrichissement** → Colle un article, l'IA compare avec ta base
- **Discussion** → Chat avec l'IA, points cles auto-ajoutes
- **Quiz** → QCM, questions ouvertes, flashcards adaptatifs
- **GitHub** → Tout est versionne et persistant
